# Evan_Cohea_FinalProject
**https://github.com/Hergifer1/Evan_Cohea_FinalProject**
**https://www.youtube.com/watch?v=3xn2VeElcQs**

My final project is a recreation of the google dinosaur game that remain a pretty faithful recreation. The player run automatically and must jump and duck to avoid obstacles. The only key difference is the addition of coins that add to your score.

Most of the files in the src folder are the different sprites for the dinosaur, pterodactyles, cacti, and coins. Three mp3 files are also included to provide sound effect for jumping, colliding with obstacles, and collecting coins. Of course there is also the python file, project.py.

I had a lot of fun making this, it posed several challenges to me like properly animating the player dinosaur and moving the obstacles across the screen. I'm not much of an artist so it's good that the sprites could all be so simplistic. I added the coin feature because I needed to have at least three class functions and I figured collecting coins while dodging obstacles would be a nice added challenge. As simple as it is, I'm glad to have made a full game for the first time.
